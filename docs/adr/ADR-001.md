# ADR-001: Azure-Only AI Inference Policy

- Status: Accepted
- Date: 2026-03-05
- Owners: Architecture Council, Compliance Lead
- Related: docs/architecture/public-baseline.md Sections 2 and 5, AGENTS.md Section 2

## Context
The platform requires strict governance, auditability, and tenant-safe inference for zero-hallucination ESG reporting.

## Decision
Production AI inference is restricted to:
- Azure AI Foundry
- Azure OpenAI

No other model providers are allowed in production paths.

## Consequences
- Strong policy and endpoint control
- Simpler legal and security review
- Reduced provider flexibility

## Verification
- Config scans must fail if non-Azure model endpoint variables are introduced.
- Integration tests must route all model calls through Azure clients.
