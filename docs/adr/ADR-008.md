# ADR-008: Cost Governance and Budget Controls

- Status: Accepted
- Date: 2026-03-05
- Owners: Platform Operations, Finance
- Related: docs/architecture/public-baseline.md Sections 6 and 8, AGENTS.md Section 11

## Context
Agent loops and retrieval depth can increase cost unpredictably.

## Decision
FinOps controls:
- Budget policy per tenant and run class
- Token budgets and retrieval-depth ceilings
- Cost telemetry by run and by agent

## Consequences
- Predictable run economics
- Possible quality trade-off if budgets are too strict

## Verification
- Cost metrics must be emitted for all run stages.
- Policy engine must reject runs that exceed hard budget limits.
