# ADR-003: OCR and Document Intelligence Default

- Status: Accepted
- Date: 2026-03-05
- Owners: Data Engineering, Compliance
- Related: docs/architecture/public-baseline.md Sections 4 and 5

## Context
Evidence ingestion must be high accuracy and auditable across invoices, forms, and scanned records.

## Decision
Primary OCR and layout extraction:
- Azure AI Document Intelligence

Exception path:
- Local OCR remediation (ocrmypdf) for controlled edge cases only

## Consequences
- Standardized cloud-first extraction
- Controlled exception handling for difficult scans

## Verification
- OCR pipeline tests must pass using Azure Document Intelligence client.
- Exception path must require explicit remediation flag.
